## What this project is

Liberty Basketball Analysis is a Flask + SQLite app for basketball operations. It supports schedules, games, sources, film review, AI-assisted event generation, practice workflows, runtime telemetry, and debug/issue tracking.

## Core runtime facts

- main app: `app.py`
- schema source of truth: `schema.sql`
- default DB: `film_analysis.db`
- upload folder: `uploads/`
- standalone start: `python app.py`
- tests: `.venv/bin/python -m pytest tests/ -q`

## Deployment surfaces

- standalone install: `scripts/setup_project.py`
- transfer bundle: `scripts/build_transfer_bundle.sh`
- CPU container: `docker-compose.yml`
- GPU container override: `docker-compose.gpu.yml`
- service template: `deploy/liberty-basketball-analysis.service`

## Configuration and runtime environment

Environment variables:

- `LIBERTY_DATABASE`
- `LIBERTY_UPLOAD_FOLDER`
- `PORT`
- `LIBERTY_DEBUG`

Feature flags live in `config.py`.
